# Analysis Checklist

## 常见核查项目

开始分析前先检查：

1. 指标定义是否唯一
2. 分子分母是否来自同一母体
3. 时间字段是否统一
4. 标签是否成熟
5. 是否有策略切换或口径切换
6. 是否有样本污染或回填

## 推荐拆解顺序

1. 先看整体趋势
2. 再看时间切片
3. 再看核心分群
4. 最后看规则、评分、操作链路

## 风控场景常用输出

- 通过率异常拆解
- 首逾或 30+ 上升归因
- 欺诈命中率与损失率排查
- 渠道质量对比
- 策略上线前后效果对比
- Vintage 表现差异分析

## 输出模板

1. 目标指标与口径
2. 样本范围与观察窗口
3. 核心拆解维度
4. 已发现异常
5. 当前不能下结论的地方
6. 建议补充分析项
